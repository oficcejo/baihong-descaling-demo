
## 动画说明

动画展示了以下过程：
1. 除垢灵分子随水流方向由左向右流动
2. 分子间的范德华力被打破（白色连接线）
3. 水垢分子被除垢灵分子吸附并带走
4. 按照由内到外的顺序清除水垢：
   - 第一遍清除内层水垢
   - 第二遍清除中层水垢
   - 第三遍清除外层水垢
   - 第四遍清除残余水垢
